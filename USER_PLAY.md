# 丛林大冒险：玩法说明

## 启动方式
- 桌面版：在项目根目录执行 `./gradlew :desktopApp:run`（Windows PowerShell 可用 `gradlew.bat :desktopApp:run`）。
- Android：用 Android Studio 打开工程，运行 `androidApp`。

## 基本玩法流程
1. 启动后先在“选择角色”区域挑选可用角色（带“可选/已选择”标签）。
2. 进入“当前事件”，阅读事件描述与提示。
3. 在“可选行动”中点击选项按钮做出选择，或点击“继续前进”。
4. 若事件为战斗，会自动结算战斗并显示战斗日志。
5. 回合推进会改变章节与关卡进度，直到进入新的章节或关卡。

## 界面与操作说明
- 顶部栏
  - 显示标题、回合数、章节与关卡进度，以及“最近操作”提示。
- 选择角色
  - 左侧角色列表可滚动浏览；右侧展示详细属性与技能说明。
  - “使用角色”按钮可快速切换到当前角色。
- 当前事件
  - 显示事件标题、类型、难度、节点信息与事件描述。
- 事件日志
  - 记录当前冒险过程，包括事件文本、战斗过程、奖励结算等。
- 可选行动
  - 点击事件提供的按钮作出选择；“继续前进”用于推进到下一个节点。
- 快捷面板
  - “状态/装备/背包”可切换右侧面板展示。
  - 目前装备与背包为占位展示，状态面板可查看角色生命、能量与资源。
- 存档管理
  - 共有 8 个存档槽。
  - 点击“保存”写入当前进度；点击“读取”恢复存档（槽位为空时无法读取）。

## 战斗说明
- 遇到战斗事件后系统自动结算，无需手动操作技能。
- 战斗胜利会根据事件结果获得奖励；失败会保留部分资源并继续推进。
- 战斗过程与结果会写入“事件日志”。

## 小提示
- “事件日志”会记录关键变化（如 HP/金币变化、战斗结果）。
- 若想保留进度，建议在关键节点使用存档槽。
